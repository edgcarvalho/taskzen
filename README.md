TaskZen
TaskZen é uma plataforma de produtividade que une eficiência e saúde mental. Ela ajuda os usuários a organizar suas tarefas de forma equilibrada, evitando sobrecarga e priorizando o bem-estar.

⚡ Funcionalidades Atuais
Cadastro de Tarefas: Adicione tarefas com prioridade e tempo estimado.

Alertas de Sobrecarga: Receba notificações quando adicionar muitas tarefas para um único dia.

🚧 Funcionalidades Futuras
Check-ins de Humor: Registre seu humor diariamente (com emojis).

Modo Zen: Foco total em uma tarefa por vez, ocultando as demais.

Relatórios Semanais: Insights sobre o tempo produtivo e o seu humor ao longo da semana.

Edição e Exclusão de Tarefas: Editar, excluir ou marcar tarefas como feitas.

🛠 Tecnologias Usadas
Frontend: React.js, Vite, Tailwind CSS

Backend: Em desenvolvimento (Node.js + Express)

Banco de Dados: Em desenvolvimento (MySQL, Prisma ORM)

Autenticação: Firebase Auth

🌱 Objetivo do Projeto
O TaskZen busca equilibrar produtividade e saúde mental. A ideia é ajudar os usuários a se organizarem sem se sobrecarregar, proporcionando uma experiência simples e focada no bem-estar.
